Expand description

Reads and parses rpeg data from a file, or stdin if no file path is given.

Returns a Result<tuple, String> where the tuple contains, in order:

  1. A Vec<[u8; 4]> containing the compressed data bytes of the image
  2. A usize for the width of the image
  3. A usize for the height of the image

ยงErrors

  • If there is an error reading from the provided file or stdin
  • If the rpeg data is invalid (ex: malformed header, mismatched size, etc.)
